KOL_sim::Mass Class Reference

#include <mass.hpp>

Mass class denotes points in space that have position and weight (in kg). The class has a 1x3 (x,y,z) std::vector for each following property: position, acceleration, velocity, external forces. These are the properties which are used to calculate the disposition of the mass.
