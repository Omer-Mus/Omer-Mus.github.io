KOL_sim::Spring Class Reference

#include <spring.hpp>

Springs are the “substance” attaching the masses together. Its force is a result of the k constant and the disposition of the masses. Spring's rest length is defined by the distance between its two masses.

void KOL_sim::Spring::actuate_spring ( double  theta)
inline
This function ignores the spring forces and actuates the spring. That is, making the spring move according to some function; the current code is an example of an actuation function defined as such: { L0 = defLength + sin(theta)*0.5; } where theta is the simulation’s time step (DT).
